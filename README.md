Miwok App
===================================

This app displays lists of vocabulary words for the user to learn the Miwok language.
Developed in a Udacity course.

Skills Used: <br />
1- OOP Programming <br />
2- View Pager <br />
3- Fragments <br />
4- Tab Layout <br />
5- Audio <br />
6- List View <br />
